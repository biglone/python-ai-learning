## 学习路径

### 第一阶段：Python 基础（建议 2-3 周）

从零开始学习 Python 编程语言：

1. **环境搭建与入门** - Python 安装、Jupyter Notebook 使用
2. **变量与数据类型** - 整数、浮点数、字符串、布尔值
3. **运算符与表达式** - 算术、比较、逻辑运算
4. **条件语句** - if、elif、else
5. **循环语句** - for、while、break、continue
6. **函数** - 定义、参数、返回值、lambda
7. **列表与元组** - 创建、索引、切片、方法
8. **字典与集合** - 键值对、集合运算
9. **字符串处理** - 方法、格式化、正则表达式
10. **文件操作** - 读写、CSV、JSON
11. **异常处理** - try-except、自定义异常
12. **模块与包** - 导入、标准库、pip
13. **面向对象编程** - 类、继承、多态

### 第二阶段：Python 进阶（建议 1 周）

提升 Python 编程技能：

1. **列表推导式与生成器** - 高效的数据处理
2. **装饰器** - 函数增强
3. **上下文管理器** - 资源管理
4. **多线程与多进程** - 并发编程
5. **异步编程** - async/await
6. **类型注解** - 代码可读性与类型检查

### 第三阶段：数据科学基础（建议 1-2 周）

掌握数据处理和可视化：

1. **NumPy** - 数组操作、数学运算
2. **Pandas** - 数据清洗、分析
3. **Matplotlib** - 基础数据可视化
4. **Seaborn** - 高级统计可视化
5. **数据清洗实战** - 处理真实世界数据
6. **探索性数据分析 (EDA)** - 数据探索方法论

### 第四阶段：机器学习基础（建议 2-3 周）

入门机器学习：

1. **机器学习概述** - 概念、流程、Scikit-learn
2. **线性回归** - 预测连续值
3. **分类算法** - 逻辑回归、决策树、随机森林
4. **聚类算法** - K-Means 聚类
5. **模型调参** - 交叉验证、网格搜索
6. **特征工程** - 特征选择与构造
7. **模型部署** - 模型持久化
8. **支持向量机** - SVM 分类与回归
9. **集成学习进阶** - XGBoost、LightGBM、CatBoost
10. **降维技术** - PCA、t-SNE、UMAP
11. **时间序列预测** - ARIMA、Prophet

### 第五阶段：深度学习基础（建议 2-3 周）

了解深度学习：

1. **深度学习入门** - 神经网络、PyTorch
2. **CNN 卷积神经网络** - 图像处理
3. **RNN 循环神经网络** - 序列数据
4. **图像分类实战** - 完整项目
5. **NLP 自然语言处理** - 文本处理基础
6. **Transformers** - 注意力机制与预训练模型
7. **目标检测** - YOLO、Faster R-CNN
8. **生成对抗网络** - GAN 图像生成
9. **强化学习入门** - Q-Learning、DQN
10. **大语言模型应用** - Prompt Engineering、RAG、Agent

### 第六阶段：实战项目

综合应用所学知识：

1. **房价预测完整项目** - 端到端机器学习流程
2. **图像分类 Web 应用** - 从训练到部署

## 环境配置

### 安装 Python

推荐使用 Anaconda：
```bash
# 下载 Anaconda: https://www.anaconda.com/download
# 或使用 Miniconda: https://docs.conda.io/en/latest/miniconda.html
```

### 创建虚拟环境

```bash
# 创建环境
conda create -n ai-learning python=3.10

# 激活环境
conda activate ai-learning
```

### 安装依赖

#### 必装依赖（核心库）

```bash
# 基础库
pip install numpy pandas matplotlib seaborn scipy

# 机器学习
pip install scikit-learn joblib

# 深度学习
pip install torch torchvision tqdm

# 图像处理
pip install Pillow

# Jupyter
pip install jupyter notebook
```

#### 按章节可选依赖

```bash

```

## 学习建议

1. **循序渐进**：按照目录顺序学习，不要跳过基础内容
2. **动手实践**：每个 notebook 都包含练习题，务必完成
3. **多做笔记**：在 notebook 中添加自己的注释和总结
4. **项目实战**：学完每个阶段后，尝试做一个小项目
5. **坚持学习**：每天保持 1-2 小时的学习时间
